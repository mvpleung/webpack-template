动态导入支持

  1. 安装依赖

     yarn add @babel/plugin-syntax-dynamic-import -D

  2. .babelrc 文件增加配置

     {
   "presets": [],
   "plugins": [
     "@babel/plugin-syntax-dynamic-import",
   ]
 }
